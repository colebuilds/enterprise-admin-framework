# archived-local-skill-b

## 状态

该 skill 已归档，仅占位保留，不用于正常调用。

## 目标

把执行结果输出为稳定格式，方便复核、追溯与交接。

## 适用场景

- 任务执行完成，需要提交结果
- 需要明确是否有计划外改动
- 需要给出下一步验证建议

## 输入要求

- 实际修改文件列表
- 实际执行命令与结果
- 是否存在计划外动作

## 执行步骤

1. 汇总实际修改文件与摘要
2. 汇总实际命令与关键结果
3. 标注新增/删除/重命名
4. 完成偏差审计（计划外改动/命令/未完成项/风险）
5. 产出验证建议（页面、流程、命令）

## 风险与禁止项

- 不得隐去计划外动作
- 不得省略失败/未完成项
- 不得只给结论不给证据

## 输出模板

- 实际修改文件：路径 + 摘要 + 是否计划内
- 实际执行命令：命令 + 结果摘要 + 是否计划内
- 文件变化：新增/删除/重命名（无则写无）
- 偏差审计：计划外改动/计划外命令/未完成项/风险点
- 建议验证：页面/流程/命令/关注点
