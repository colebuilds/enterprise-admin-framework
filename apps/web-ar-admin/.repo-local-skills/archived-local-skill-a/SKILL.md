# archived-local-skill-a

## 状态

该 skill 已归档，仅占位保留，不用于正常调用。

## 目标

将原始任务描述快速整理为可执行草案，避免直接开工导致范围失控。

## 适用场景

- 用户刚提出需求，还没有清晰边界
- 需要先产出结构化执行草案
- 希望减少来回澄清成本

## 输入要求

- 任务目标（至少一句）
- 可接受的修改范围（若有）
- 禁止触碰范围（若有）

## 执行步骤

1. 提炼目标与不包含项
2. 列出允许修改与禁止修改范围
3. 列出计划文件与计划命令（分低风险/高风险）
4. 标出待确认项（权限、依赖、风险动作）
5. 输出“确认后可执行”的简版计划

## 风险与禁止项

- 未获明确执行授权前不落盘
- 不默认放宽文件权限、依赖权限、高风险命令权限
- 不顺手扩改无关模块

## 输出要求

按以下结构输出：

- 任务目标
- 本次不包含
- 允许修改
- 禁止修改
- 计划执行
- 待确认项
- 风险提示
