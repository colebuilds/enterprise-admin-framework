# 公共能力入口索引

## 1. 目标

提供一份轻量级共享能力入口地图，帮助开发前初始化时优先定位“该先去哪里找”，而不是做全仓库无边界扫描。

本文档是**入口索引**，不是公共组件、公共函数或共享能力的全量清单。

## 2. 使用原则

- 优先从稳定入口找现成能力，不先全仓库关键词漫游。
- 找不到时，再看同业务域相似模块中的局部复用实现。
- 修改共享能力前，先确认旧调用点、兼容范围和回归影响。
- 文档只列高频、稳定、对初始化最有价值的入口。

## 3. 组件入口

### 3.1 全局组件统一出口

- `src/components/index.ts`
- 用途：优先确认当前仓库已统一导出的共享组件能力。

### 3.2 表格与表单组件

- `src/components/ProComponents/`
- 高频能力：
  - `ProCrudTable`
  - `ProForm`
  - `ProSearchForm`
- 适用场景：标准列表页、搜索表单、后台管理页。

### 3.3 表格辅助组件

- `src/components/Table/`
- 高频能力：
  - `TableAction`

### 3.4 字典组件

- `src/components/DictSelect/`
- 高频能力：
  - `DictSelect`
  - `useDictionary`

## 4. hooks 入口

### 4.1 全局 hooks 统一出口

- `src/hooks/index.ts`

### 4.2 高频 hooks

- `src/hooks/`
- 常见能力方向：
  - 权限：`usePermission`
  - 商户/时区：`useTenantOptions`

## 5. 工具与状态入口

### 5.1 通用工具

- `src/utils/`
- 常见能力方向：
  - 金额格式化
  - 时间处理
  - 请求层辅助

### 5.2 用户与权限状态

- `src/store/modules/user.ts`
- 常见能力方向：
  - 当前用户
  - 权限
  - 静态字典缓存

### 5.3 动态字典状态

- `src/store/modules/dynamicDictionary.ts`
- 常见能力方向：
  - 动态字典加载
  - 业务枚举缓存

## 6. 出款域共享入口

### 6.1 withdrawOrder 一级共享入口

- `src/views/finance/withdrawOrder/shared.ts`
- 适用场景：
  - 一级 Tab 切换
  - 跨模块预设状态透传
  - 看板跳记录页等共享上下文

### 6.2 出款看板共享上下文

- `src/views/finance/withdrawOrder/dashboard/useDashboardContext.ts`
- `src/views/finance/withdrawOrder/dashboard/useDashboardPage.ts`
- 适用场景：
  - 看板子 Tab 状态
  - 看板卡片预设
  - 商户范围与跳转状态透传

## 7. 初始化建议

开发前初始化时，建议按以下顺序找共享能力：

1. `AGENTS.md`
2. `src/components/index.ts`
3. `src/hooks/index.ts`
4. `src/utils/`
5. `src/store/modules/`
6. 目标业务域的 `shared.ts` / `context` / `useXxxPage`
7. 同业务域相似模块

如果一级入口和同域相似模块都未命中，再考虑新增局部实现或评估是否需要沉淀为共享能力。
