# 使用说明

用于在 `/new`、跨人协作、跨天续做时，快速恢复任务现场。

如果你现在还没决定“该先看哪个入口、该怎么恢复任务”，先看：

- `documents/guides/development/AI技能调用一页口令卡.md`
- `documents/guides/development/AI会话上下文恢复执行指南.md`

建议做法：

- 每个任务一份交接文档
- 若是项目正式文档，放在对应需求目录；若是本地恢复快照，放在 `/.local-work/session-context/`
- 文件名建议：`<task-slug>-YYYY-MM-DD.md`
