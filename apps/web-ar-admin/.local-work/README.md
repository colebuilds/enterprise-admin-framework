# .local-work

本目录用于存放**执行期、本地态、非业务归档**的工作文档与产物。

## 默认用途

- `plans/`
  - 通用执行阶段实施计划
  - PIR / superpowers / 其他 agent 产出的计划文件
  - 临时执行清单
- `permission-audits/`
  - 模块级权限静态预审
  - 模块级权限权威核对记录
  - 单次权限映射表与疑似问题清单
  - 可复用的静态预审 / 权威核对模板
- `session-context/`
  - 会话恢复快照
  - 最新上下文指针
- 其他本地产物
  - 临时验证记录
  - 本地实验草稿
  - 非正式截图与对比结果

## 兼容路径

- `superpowers/plans/`
  - 历史兼容目录
  - 旧会话、旧脚本或旧 agent 可能仍会读取这里
  - 新计划默认不要再写入该目录，统一写入 `plans/`

## 不应放入本目录的内容

- 需求说明
- 技术设计
- 长期测试方案
- 架构决策
- 规范与指南

以上正式项目文档应按职责进入 `documents/`。

## 归档原则

- 这里的文件默认视为执行期工作文档，不替代正式业务文档
- 若某份内容后续需要长期沉淀，应整理后再提升到 `documents/`
